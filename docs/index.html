Blockchain Research Papers Network Analysis

We are going to analyze the papers and citations data we have. Raw data comes in csv format, however we have preformatted the data to suite visNetwork requirement. ‘Edges.RData’ has two columns, to and from, showing the connections between nodes. ‘Nodes.RData’ has three columns; id, label, and group. Id numbers the papers from 1 to the number of rows, the label column has the Web of Science identifiers for the papers. Interestingly, we have divided the papers in groups based on their year of publication. Some NAs were removed to run a smooth and fruitful network graph.
